# 💼 HRConnect

> A comprehensive Human Resource Management System (HRMS) that streamlines employee management, attendance, leave requests, payroll, holidays, and administrative workflows through a modern web interface.

---

## 📖 Overview

HRConnect is an enterprise-style HR management platform designed to simplify daily HR operations. The application provides centralized management of employees, attendance, leave, payroll, holidays, and organizational information through an intuitive and responsive interface.

This project was developed collaboratively as a team project.

# 🚀 Development Setup

## Prerequisites

- Node.js (v18+)
- npm or yarn

## Clone Repository

```bash
git clone https://github.com/bknath/hrconnect.git

cd hrconnect
```

## Install Dependencies

```bash
npm install
```

## Environment Variables

Create a `.env` file:

```env
DATABASE_URL=
SESSION_SECRET=
```

## Start Development Server

```bash
npm run dev
```

Open:

```
http://localhost:5000
```

# 🧪 Local Development

When running locally, the application automatically:

- Uses local authentication
- Uses memory sessions
- Can run without a database
- Supports full HRMS functionality
